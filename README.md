# shoot Game #

----------
项目描述:
一个使用c#和unity5 实现的3D TPS游戏 

功能描述：

	1. 游戏场景制作
		1.1 3D模型、纹理、材质
		1.2 墙体、天空、光照
	2. 游戏角色制作
		2.1 玩家角色
			2.1.1 角色移动、摄像机追踪、动画片段、实时阴影
		2.2 敌对角色
			2.2.1 怪兽攻击例程、受击反应、血迹效果
	3. 碰撞感知、子弹发射效果、油桶爆炸效果
